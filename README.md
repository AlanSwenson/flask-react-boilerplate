## Install

This is install instruction for the boilerplate, if you want to start from scratch follow the presentation instead:

You need:

- Python 3
- NodeJS
- Yarn

`pip install pipenv`
`pipenv shell`
`pipenv install`

## Run

`pipenv shell`
`make backend`

In a separate terminal
`make frontend`
